# Nextee tools

![nextee](https://github.com/mitch1009/nextee/blob/main/public/logo.svg?raw=true)
## Getting Started

```bash
pnpm install
#  create env variables
#  copy .env.example to .env
AUTH_AZURE_AD_ID=
AUTH_AZURE_AD_SECRET=
AUTH_AZURE_AD_TENANT=
AUTH_SECRET=
AUTH_URL=
AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
#  Run dev server
pnpm dev
```
